# CodSoft Data Analytics Internship

Welcome to my repository for the **CodSoft Data Analytics Internship**. This repository contains my clean data pipelines, implementations, and visual insights for the three assigned internship tasks.

## 📂 Repository Structure
The project files are organized directly at the root for clear navigation:
*   `codsoft_task1/` - Implementation and scripts for Task 1.
*   `codsoft_task2/` - Implementation and scripts for Task 2.
*   `codsoft_task3/` - Implementation and scripts for Task 3.
*   `sample_data.csv` - The raw source dataset.
*   `cleaned_data.csv` - Transformed dataset after data cleaning workflows.
*   `age_boxplot.png` & `city_distribution.png` - Key analytical visualization plots.

---

## 🛠️ Tasks Overview

### 📊 Task 1: Data Exploration and Cleaning
*   **Objective:** Import the raw dataset, analyze structural missingness, remove duplicate anomalies, and enforce proper data types.
*   **Key Deliverable:** Generated the `cleaned_data.csv` pipeline file.

### 📉 Task 2: Exploratory Data Analysis (EDA)
*   **Objective:** Uncover core trends, statistical behaviors, and mathematical correlations across features like Age and City distributions.
*   **Key Deliverables:** Extracted distribution statistics and structured the analysis profiles.

### 🖼️ Task 3: Data Visualization
*   **Objective:** Translate data distributions into standalone graphical assets to communicate insights effectively to stakeholders.
*   **Key Deliverables:** Visualized data variations with `age_boxplot.png` and `city_distribution.png`.
